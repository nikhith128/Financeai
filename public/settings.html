Settings

Backup Your Data

This app stores data as plain JSON files on the server. On free hosting tiers (like Render's free plan), those files can be wiped whenever the server restarts or redeploys. Download a backup regularly, and keep it somewhere safe (your phone, a cloud drive) so you can restore it if that happens.

Restore From Backup

Choose a backup file you downloaded earlier. This replaces all current data (transactions, budgets, and goals) with what's in the file — anything logged since that backup was taken will be lost.


Reset All Data

Wipes every transaction and goal, and resets budgets back to their defaults. This cannot be undone unless you have a backup file to restore from.
